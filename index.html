HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element defines the CSS styles associated with a website.
  • The base element defines the base URL for a webpage.
  • The link element connects an external resource to the HTML document.
  • The script element embeds client-side scripts in the HTML document.
  • The Meta element defines metadata such as the character set, description, keywords, author, and viewport.
  • YOU SHOULD INCLUDE THE FOLLOWING META ELEMENT IN ALL YOUR WEB PAGES, SETTING THE VIEWPORT TO MAKE YOUR WEBSITE LOOK GOOD ON ALL DEVICES: meta name="viewport" content="width=device-width, initial-scale=1.0"

CSS

  • A margin indicates how much space we want around the outside of an element, not visible on page, just adds buffer, 0 = no space
  • A padding indicates how much space we want around the content inside an element. Increasing padding makes entire element bigger
  • Three ways to style are Inline, Internal Style Sheet, External Style Sheet (Preferred)
  • SEPARATION OF CONCERNS (SoC): refers to separating code into different files based on use https://en.wikipedia.org/wiki/Separation_of_concerns
  • GOOD ARTICLE ON CSS BOX MODEL: https://developer.mozilla.org/en-US/docs/Learn/CSS/Building_blocks/The_box_model
  • In simple terms: CSS is like telling the computer how to decorate stuff on the page. Selectors are like pointing to the thing you want to decorate. Classes are like name tags for a group of things that all get the same decoration.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • GIT FLOW: Add, commit, and push work to the feature branch. Then, open a pull request, merge changes with the main branch, and close the issue

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Arrays are zero-indexed and sequential. This means that the first item in any array has an index of 0, not 1
  • A Function is a set of instructions that tells the computer how to perform a certain task
  • JavaScript is also a weakly-typed language, which means it allows implicit type conversion when an operation involves mismatched types, instead of throwing type errors
